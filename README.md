# Linux & Cyber Security Basics Lab

Welcome to my repository for the first Cyber Security Lab assignment.

### 🎯 Overview of Tasks Completed
In this lab, I established a virtualized Linux environment and practiced essential file management techniques. The core focus areas were:
- **Environment Check:** Validating the Ubuntu kernel and OS details.
- **Directory Traversal:** Using commands like `cd`, `pwd`, and `ls` to move around the system.
- **Workspace Construction:** Creating a nested directory tree for course modules (Linux, Networking, CyberSecurity, EthicalHacking).
- **File Manipulation:** Generating text documents and utilizing `cp` and `mv` commands to consolidate reports.

### 📂 Repository Contents
The repository mimics the directory structure built during the lab:
```text
CyberSecurity_Lab/
├── CyberSecurity/       # Security principles & firewall commands
├── EthicalHacking/      # Penetration testing basics & tools
├── Linux/               # Core OS notes & CLI commands
├── Networking/          # OSI model & packet routing commands
└── Reports/             # Centralized location for all module reports
```

Each subdirectory contains custom `.txt` files detailing command summaries, technical notes, and individual lab execution reports.
